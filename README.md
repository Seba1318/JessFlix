# JessFlix 🎬📚

**"Read the Story. Watch the Vision."**

JessFlix is a sleek, digital gallery built for storytelling enthusiasts. It brings together the essentials of film, television, and literature into one elegant, single-page interface. Instead of just browsing, JessFlix encourages users to discover and appreciate the art of the story across different media formats.

---

## ✨ Features

- **Multi-Media Catalog:** A curated collection of Movies, TV Shows, and Books.
- **Single Page Application (SPA):** Seamless navigation between categories without page reloads, powered by Vanilla JavaScript.
- **The Logbook:** A personal tracker (under development/integrated) to keep record of your journey through various stories.
- **Dynamic Content:** Interactive details windows and a responsive search functionality.
- **Polished UX:** Smooth "float-in" scroll animations and a modern, minimalist design aesthetic.
- **Modular Architecture:** Cleanly separated concerns with ES Modules, organized by functionality and data.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure for the web.
- **CSS3:** Custom styles with a focus on component-based architecture and responsiveness.
- **JavaScript (ES6+):** Pure Vanilla JS using Modules for state management and DOM manipulation.
- **Assets:** WebP optimized images for fast loading and crisp visuals.

## 🎨 Design Philosophy

JessFlix is designed to be more than just a media list; it's a "digital gallery." The focus is on high-quality visuals, clear typography (using Petit Formal Script), and a layout that lets the content breathe.
